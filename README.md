# Swiggy vs Zomato: Restaurant Profitability Dashboard

## Overview
An interactive Power BI dashboard analyzing 2,500 restaurants across 10 Indian cities 
to help restaurant owners choose between Swiggy and Zomato based on profitability data.

## The Question
Which food delivery platform yields better profit for a restaurant in India? 
The answer depends on location, cuisine type, business model, and price point.

## The Solution
A 6-panel interactive dashboard with real-time filtering:

## Dashboard Preview
📊 View the full dashboard: [Dashboard - Zomato Vs Swiggy.pdf](Dashboard%20-%20Zomato%20Vs%20Swiggy.pdf)


### Panels:
1. **Headline KPI** — Which platform wins on average
2. **Average Monthly Net Profit** — Side-by-side comparison
3. **Average Monthly Orders** — Volume comparison
4. **Platform Winner by City** — Geographic map showing which platform dominates each city
5. **Delivery Time vs Rating** — Scatter plot analyzing the correlation
6. **Profit Difference by Restaurant Type** — Segmentation analysis

## Key Findings
- 📊 Platforms nearly tied globally (₹404/month difference = 0.3% margin)
- 🗺️ Geographic variation is significant — Mumbai patterns differ from Delhi
- 🏢 Business model matters — Cloud Kitchen vs Fine Dine favor different platforms
- ⚡ Delivery speed weakly predicts ratings — speed alone doesn't guarantee high ratings
